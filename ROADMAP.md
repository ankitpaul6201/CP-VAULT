# CP Vault Roadmap

This document outlines the planned development trajectory for CP Vault.

> **Note**: Timelines are approximate and subject to change based on community feedback and contributor availability.

---

## ✅ v1.0.0 — Released

**Theme**: Foundation & Core Platforms

- [x] LeetCode support (GraphQL intercept)
- [x] Codeforces support (DOM polling)
- [x] CodeChef support (API intercept)
- [x] HackerRank support (API intercept)
- [x] GitHub OAuth 2.0 authentication
- [x] Atomic multi-file commits via Git Database API
- [x] Auto-generated per-problem README.md
- [x] Auto-updated root repository README with statistics
- [x] Duplicate detection via content hashing
- [x] Retry queue for failed syncs
- [x] Desktop notifications (success/failure)
- [x] Dark/Light theme with system detection
- [x] Streak tracking
- [x] Docker-based OAuth backend
- [x] GitHub Actions CI

---

## 🔜 v1.1.0 — More Platforms

**Theme**: Expanding Platform Coverage

- [ ] AtCoder support
- [ ] CSES support
- [ ] SPOJ support
- [ ] GeeksForGeeks support
- [ ] Custom commit message templates (full UI)
- [ ] Export solutions as ZIP file
- [ ] Improved error messages and diagnostics
- [ ] Settings import/export

---

## 🔜 v1.2.0 — Statistics & Insights

**Theme**: Analytics & Tracking

- [ ] Statistics dashboard with charts (problems/day, language breakdown, platform distribution)
- [ ] Weekly/monthly progress reports
- [ ] Solution tagging system (`#dp`, `#graph`, `#greedy`)
- [ ] Calendar heatmap (like GitHub contributions)
- [ ] Multi-repository support (separate repos per platform)

---

## 🔜 v1.3.0 — Developer Experience

**Theme**: Power User Features

- [ ] Time & space complexity placeholders auto-filled
- [ ] Approach notes template in generated README
- [ ] Batch sync (sync multiple solutions at once)
- [ ] Sync history export (CSV/JSON)
- [ ] Keyboard shortcuts

---

## 🔮 v2.0.0 — Future Vision

**Theme**: AI & Cross-Browser

- [ ] Firefox extension support
- [ ] AI-generated approach notes in problem READMEs (via Gemini/GPT)
- [ ] Portfolio website generator from your solutions repo
- [ ] Collaboration features (shared vaults / team challenges)
- [ ] Chrome Web Store publication
- [ ] Mobile companion app (view stats)

---
