# Security Policy - CP Vault

We take the security of CP Vault seriously. If you believe you have found a security vulnerability, please report it to us using the instructions below.

## Reporting a Vulnerability
- **Do NOT open a public GitHub issue** for security vulnerabilities.
- Please email security reports to **ankitpaul9341@gmail.com** (or your designated security contact email).
- Include the following details in your report:
  - Description of the vulnerability and its potential impact.
  - Step-by-step instructions (or a proof-of-concept script) to reproduce the issue.
  - Details of the environment (browser version, OS, extension version) where the issue was tested.

## Our Security Response Commitment (SLA)
- **Response time**: We will acknowledge receipt of your report within 48 hours.
- **Triage and fix**: We aim to resolve critical vulnerabilities within 7 business days and push verified patches immediately.
- **Disclosure**: We will coordinate the public announcement of fixed vulnerabilities with you to ensure a responsible disclosure timeline.
